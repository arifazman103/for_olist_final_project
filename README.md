# For Olist Final Project

> 🚧 **Work in Progress** – Full pipeline coming soon.

## Planned Scope

An end-to-end machine learning pipeline built on the
[Olist Brazilian E-Commerce Public Dataset](https://kaggle.com).

### Problem Statement
Predict whether an order will be **delivered late** based on:
- Seller location and distance to buyer
- Product category and weight
- Historical shipping time patterns
- Logistics and carrier patterns

### Planned Tech Stack
- Python (Pandas, Scikit-learn)
- PostgreSQL / SQLAlchemy
- Feature engineering & EDA notebooks
- Model training: Logistic Regression, Random Forest, XGBoost
- Evaluation: Precision, Recall, F1-Score, ROC-AUC
